JSON data filtering & sorting App

While entering the JSON data please keep the next formatting:

                    
                    {"data": [{"user": "mike@mail.com", "rating": 20, "disabled": false},
                              {"user": "greg@mail.com", "rating": 14, "disabled": false},
                              {"user": "john@mail.com", "rating": 25, "disabled": true}],
                    "condition": {"include": [{"disabled": true}], "sort_by": ["rating"]}}

"condition" supports "include" and "exclude" keys
